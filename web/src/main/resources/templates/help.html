Frequently Asked Questions

What is the Trooper Card Generator?

The Trooper Card Generator is a fan-made tool for the Infinity tabletop game by Corvus Belli.

It converts an army code from the official Infinity Army builder into printable trooper cards with unit stats and profiles.

Why can’t I print multiple cards on one page?

This is usually a printer issue. Try generating the cards as a 9-to-1 PDF first, then print the PDF from your system’s print dialog with the option scale fit page

Try multiple layouts and browser, sometimes Firefox is better than Chrome.

How do I get my army code?

After creating your list in the Infinity Army builder, click “Generate Army Code”. You can then copy and paste that code into the Card Generator.

What’s the difference between the card styles?

Two card styles are available:

  • Color and image - DIN-A4: Full-color cards with unit artwork in the size of a A4 sheet. Can be shrunk by printing multiple on one page. Optimal for four or nine cards on a single A4 sheet.
  • Color and image - DIN-A4: Full-color cards with unit artwork in the dimension so six can be printed on a single A4 sheet.
  • Overview List - DIN-A4: A compact army list without image, designed to print on A4 sheet.
  • Color and image - DIN-A7: Full-color cards with unit artwork, designed to print nine cards per A4 sheet.
  • Color and image - US Letter: Full-color cards with unit artwork in the size of a US Letter sheet. Can be shrunk by printing multiple on one page.
  • Black and white - playing card: Minimalist, ink-friendly cards sized at 59 × 91 mm (standard playing card size). Not all options work with this style.
What does the “Unit Filter” do?

Use the “Distinct Units” option to generate only one card per unique trooper profile. The “No Filter” option creates a card for every trooper in your list, even if they are identical.

Why are some images missing?

Currently Corvus Belli only provides images for certain unit profiles. The community is working to add images for all troopers. If you’d like to help, please contact me.

Why is there a * next to some PS or Range MOD values in the weapon table?

The asterisk (*) indicates that modifiers from skills, equipment or profile have altered the weapon’s default value.

What means the C, E or S at the end of a saving roll expression?

C means the weapons does Continous Damage, E that the weapon does E/M damage and S that the weapon does Shock damage

Why do I get an error message for my army code?

Your army code might be incomplete or outdated. When Corvus Belli updates their data, older codes can become invalid.

First, check if your code still works in the Infinity Army tool. If it does, the issue might be a bug—please email me or open an issue on GitHub.

Why is the layout broken for one specific trooper?

Some unit profiles have unusual or too much data and that can cause layout issues. Try the different options for this trooper to find the best printable option. Sometimes you need to remove the image to get all weapons.

Please email me or open an issue on GitHub so I can improve the formatting.

You can email me directly or create an issue on GitHub to suggest new features or improvements.
